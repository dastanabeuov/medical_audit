# Папка для кодов МКБ

Поместите сюда файлы с кодами МКБ-10 в форматах:
- PDF
- DOCX
- XLSX
- TXT

Формат данных:
```
A00.0 - Холера, вызванная холерным вибрионом 01, биовар cholerae
A00.1 - Холера, вызванная холерным вибрионом 01, биовар eltor
...
```

После добавления файлов выполните команду:
```bash
rails knowledge_base:import_mkb
```

Для полного обновления базы:
```bash
rails knowledge_base:refresh_all
```
